HD 的工具箱

趁手的前端小工具，
一個個收進工具箱

每個工具都直接打開就能用。能在瀏覽器裡做完的，檔案就不會離開你的電腦。

PDF to JPG

把 PDF 每一頁轉成 JPG，可調解析度與品質，打包成 ZIP 下載。

圖片切片

上傳圖片，選欄數與列數，切成等分小圖，可選一併去除綠幕，打包成 ZIP 下載。

圖片去背

去掉圖片背景：綠幕模式即時去背，AI 模式去任意背景。輸出透明 PNG。

圖片壓縮

批次壓縮圖片，JPEG / PNG / WEBP / AVIF 互轉、可調品質與縮放，打包下載。

AI 貼圖生成

上傳照片、選風格、列好表情 → AI 生一整組貼圖，自動去背縮到 LINE 尺寸，打包 ZIP。串 OpenAI / Google，key 存在你瀏覽器。
